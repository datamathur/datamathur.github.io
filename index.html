Projects

Course Project - Artificial Neural Networks

In January 2021, I started working on my course project for IEE 03 - Artificial Neural Networks with my classmate Aman Arora.
The project aims to develop classifier for Classification of Breast Tumor into Malignant (cancer tumor) and Benign (non-cancer tumor) using features obtained from several cell images. The dataset we used for this purpose was Breast Cancer Wisconsin dataset from Scikit Learn. The classification process was carried out by three models, Support Vector Machine (SVM), Neural Network (with Particle Swarm Optimizer), and Neural Network (with Gradient Descent). The main objective is to compare these three models and find the most suitable model.
For more information refer to Project Report or visit the Project Repository

Facial Expression Recognition

In September 2020, I collaborated with my friend Mohit Sharma to do a comparative study of various Machine Learning and Computer Vision models over the FER2013 dataset.
Mohit worked on various classical Machine Learning models to build a Facial Expression Recognizer while I worked on various Deep Learning models to build the same. To see our work visit this repository.

Melbourne Housing Prices

In August 2020, I worked on a regression dataset, Melbourne Housing Snapshot. This project was very curious because it had 20 features in it which were highly inter-related and I selected the appropiate features without the use of techniques like PCA.
I selected 9 features to train my models - Rooms, Type, Distance, Car, Bathroom, Landsize, Lattitude, Longtitude, and Propertycount, each of which is clearly explained in my GitHub repository.
I made 3 model to train on this dataset and the R2 Score of those models were, 0.999999999999997 from Polynomial Regression, 0.9999916206262129 from Lasso Regression, and 0.9999942091830432 from Decision Tree Regressor.

CIFAR-10

Later in May 2020 after practicing my basics over Digit Recognition, I further dove into Computer Vision by trying out an immersive comparative study of various successful CNN models over the famous CIFAR-10 dataset.
I used three Deep Learning Models on this CIFAR-10 dataset, Neural Networks, Convolutional Neural Networks, and Residual Neural Networks giving accuracies of 48.42%, 72.899%, and 77.759% respectively.
By this comparative study I devised my own proof of concept that by using the concept of residual networks or skip connections, accuracy of a Computer Vision Model can be enhanced.

Digit Recognizer

After learning basics of Convolutional Neural Networks, I started working on the famous handwritten digits dataset by MNSIT in May 2020.
I used two very primitive DL model to obtain accuracies of 97.879% from Fully Connected Neural Network Model and 99.21% from Convolutoinal Neural Network Model.

Titanic

I started practicing Machine Learning with the cliche of ML Datasets, Titanic Dataset. I used Logistic Regression, Support Vector Machine(SVM), and Neural Networks to solve this dataset.
Out of various features in the dataset, I decided to work on were Passenger class, Sex, Age, and Family count (which I extracted from no. of sibling spouse and parents children).
The accuracies obtained were 81.00% from Logistic Regression, 78.77% from SVM, and 74.44% from Neural Networks. This GitHub repository to my work.
